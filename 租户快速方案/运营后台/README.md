# 模型定价管理 - 高保真原型

## 项目概述

这是一个基于Vue 3 + Element Plus + 飞书设计系统的高保真原型，用于管理AI模型的Token消耗积分规则。该原型完全按照业界顶级标准设计，具备高度的层次感和极致的用户体验。

## 功能特性

### 🎯 核心功能

- **仪表盘概览**: 实时展示模型总数、定级分布、待定价模型等关键指标
- **智能筛选**: 支持按模型名称、系列、定级等多维度筛选
- **实时编辑**: 支持表格内直接编辑，提供即时保存和取消功能
- **批量操作**: 支持批量编辑、删除，大幅提升操作效率
- **数据导入导出**: 支持CSV格式的批量导入导出功能

### 🎨 设计亮点

- **飞书设计系统**: 采用飞书设计语言，确保视觉一致性
- **响应式布局**: 完美适配不同屏幕尺寸
- **交互动效**: 丰富的动画效果和过渡动画
- **无障碍设计**: 支持键盘快捷键操作

### 🚀 技术特性

- **Vue 3 Composition API**: 现代化的响应式数据管理
- **Element Plus**: 企业级UI组件库
- **ECharts**: 专业的数据可视化图表
- **模块化架构**: 清晰的代码结构和组件分离

## 快速开始

### 1. 打开页面

直接在浏览器中打开 `model-pricing-management.html` 文件即可使用。

### 2. 功能使用

#### 仪表盘操作
- 点击"模型总数"卡片查看所有模型
- 点击"待定价模型"卡片筛选未定价模型
- 点击"经济型模型"卡片筛选经济型模型
- 查看"模型定级分布"饼图了解分布情况

#### 数据筛选
- 使用搜索框按模型名称或系列搜索
- 使用下拉菜单按定级或系列筛选
- 点击"重置"按钮清空所有筛选条件

#### 表格操作
- **单行编辑**: 点击"编辑"按钮进入编辑模式
- **批量选择**: 使用复选框选择多个模型
- **批量编辑**: 选择模型后点击"批量编辑"按钮
- **排序**: 点击列标题进行排序
- **分页**: 使用底部分页组件浏览数据

#### 数据管理
- **添加模型**: 点击"添加模型"按钮
- **导入数据**: 点击"导入配置"按钮选择CSV文件
- **导出数据**: 点击"导出配置"按钮下载CSV文件
- **刷新数据**: 点击"刷新"按钮重新加载数据

### 3. 键盘快捷键

- `Ctrl/Cmd + S`: 保存当前编辑
- `Escape`: 取消编辑
- `Ctrl/Cmd + A`: 全选（在表格中）

## 数据模型

### 模型数据结构

```javascript
{
  id: Number,              // 唯一标识
  modelName: String,       // 模型名称
  modelSeries: String,     // 模型系列
  inputCredits: Number,    // 输入积分（每1k tokens）
  outputCredits: Number,   // 输出积分（每1k tokens）
  tier: String,           // 定级（economy/standard/advanced/flagship）
  notes: String,          // 备注信息
  createTime: String,      // 创建时间
  updateTime: String       // 更新时间
}
```

### 定级说明

- **经济型 (economy)**: 高性价比模型，适合通用任务
- **标准型 (standard)**: 平衡性能和成本的模型
- **高级型 (advanced)**: 高性能模型，适合复杂任务
- **旗舰型 (flagship)**: 顶级模型，性能最强但成本最高
