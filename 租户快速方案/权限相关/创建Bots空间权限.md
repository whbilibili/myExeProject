# 专项权限 - 创建Bots空间

## 功能设计

| 功能点 | 描述 |
| :--- | :--- |
| **空间创建者角色** | 在租户的企业概览模块中，新增一个名为"空间创建者"的权限角色。该角色支持多选添加个人或部门，其交互逻辑与现有的租户管理员角色保持一致。被赋予此角色的成员或组织，将获得在Bots平台中创建空间的权限。**此为唯一的权限点。** |
| **审批流程集成** | 1.  **审批表配置**：<br>    *   **字段精简**：复用Bots平台原有的审批表，但移除"应用角色"列。<br>    *   **数据过滤**：审批列表中仅显示"待处理"的申请记录，已审批（无论通过或拒绝）的记录将自动隐藏。<br>2.  **审批联动**：<br>    *   一旦用户的空间创建申请在审批表中被**通过**，系统将自动为该用户赋予"空间创建者"的角色。 |
