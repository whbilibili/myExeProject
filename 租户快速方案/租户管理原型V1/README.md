# 租户管理原型 V1

基于租户后台管理V1.1方案的高保真原型，提供运营人员管理租户的完整界面。

## 功能特性

### 📊 租户列表页
- 数据仪表盘：租户总数、正常租户、已禁用租户、套餐到期、今日新增
- 高级搜索筛选：支持模糊搜索、版本筛选、状态筛选、时间范围、创建人/所有者
- 租户列表：展示租户详细信息，支持快速操作

### 🔍 租户详情页
包含6个详细标签页：

1. **数据总览**：完整的数据看板，包含资产、积分、Token、设备使用等多维度分析
2. **成员与部门**：组织架构树+成员列表管理
3. **权限管理**：个人和部门权限配置
4. **订阅与配额**：套餐信息和资源配额管理
5. **账单详情**：消费流水明细查询
6. **资产明细**：智能体、工作流、工具等资产盘点

## 技术栈

- **UI框架**：Element Plus 2.4.4
- **图表库**：ECharts 5.4.3
- **设计风格**：飞书设计系统
- **技术**：原生HTML + CSS + JavaScript

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可查看租户列表页
2. 点击租户名称可进入详情页
3. 详情页中通过标签页切换查看不同维度的数据
4. 所有交互都是模拟的，使用Mock数据
