### 第二部分：飞书风格UI重构

#### 1. 全新的设计系统

**颜色体系 - 飞书蓝**：
```css
/* 主色调 */
--primary-color: #3370FF;        /* 飞书标志性蓝色 */
--primary-hover: #1F5ADB;        /* 悬浮态 */
--primary-active: #1548B7;       /* 激活态 */

/* 文本颜色层级 */
--text-primary: #1F2329;         /* 主要文字 */
--text-secondary: #646A73;       /* 次要文字 */
--text-tertiary: #8F959E;        /* 辅助文字 */
--text-quaternary: #C9CDD4;      /* 占位文字 */

/* 背景色系统 */
--bg-white: #FFFFFF;
--bg-gray-1: #F7F8FA;            /* 浅灰背景 */
--bg-gray-2: #EFF0F2;
--bg-gray-3: #E5E6EB;
```

#### 2. 规范的间距系统

**4px 基础单元**：
- 4px, 8px, 12px, 16px, 20px, 24px, 28px, 32px, 40px, 48px, 64px
- 所有padding、margin、gap都遵循此系统
- 确保视觉上的节奏感和一致性

#### 3. 简约的圆角系统

```css
--radius-xs: 2px;    /* 小组件 */
--radius-sm: 4px;    /* 按钮、标签 */
--radius-md: 8px;    /* 卡片、输入框 */
--radius-lg: 12px;   /* 大卡片 */
--radius-xl: 16px;   /* 特殊容器 */
```

#### 4. 轻量的阴影系统

飞书风格强调扁平和简洁，减少阴影使用：
- **xs**: 0 1px 2px rgba(0, 0, 0, 0.05) - 极轻微
- **sm**: 0 2px 4px rgba(0, 0, 0, 0.06) - 默认
- **md**: 0 4px 8px rgba(0, 0, 0, 0.08) - 悬浮
- **lg**: 0 6px 16px rgba(0, 0, 0, 0.10) - 弹出
- **xl**: 0 8px 24px rgba(0, 0, 0, 0.12) - 模态
