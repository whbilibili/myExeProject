# 用户管理系统 - 高保真原型 V3

> 基于业界顶级标准设计的企业级用户管理后台高保真原型

## 📋 项目介绍

这是一个采用现代设计理念和技术栈构建的用户管理系统高保真原型，提供完整的用户列表管理、用户详情分析、数据可视化等功能。界面设计参考了Ant Design和飞书的设计规范，注重层次感、用户体验和视觉精致度。

## ✨ 核心特性

### 🎨 设计亮点

- **精致的视觉设计**: 统一的配色系统、图标系统、排版规范
- **层次感设计**: 多层次阴影系统、渐变背景、分组布局
- **极致用户体验**: 流畅动画、即时反馈、智能提示
- **契合用户习惯**: 参考Ant Design和飞书设计规范

### 🚀 功能模块

#### 1. 用户列表页

- **仪表盘KPI卡片**
  - 用户总数、正常用户、已禁用用户
  - 动态数字动画、渐变背景、悬浮效果

- **高级搜索栏**
  - 模糊搜索（用户名称/ID/手机/邮箱/MIS）
  - 状态筛选（全部/正常/已禁用）
  - 归属租户多选筛选
  - 创建时间范围筛选

- **数据表格**
  - 用户基本信息展示
  - 归属租户标签展示（支持悬浮查看全部）
  - 联系方式一键复制
  - 状态管理（重置密码/禁用/注销）
  - 分页器

#### 2. 用户详情页

**顶部导航**: 返回按钮、用户名称、操作按钮

**用户基础信息卡片**: 大头像、核心信息展示

**五个分析标签页**（通过iframe嵌入）:

##### 📊 数据总览

- **顶层KPI**: 活跃天数、资产调用总次数、总消耗积分、资产使用率
- **用户价值与活跃分析**
  - 活跃度日历热力图
  - 资产调用趋势（堆叠面积图）
  - 最活跃资产排行（条形图）
- **核心成本分析**
  - 积分消耗趋势（堆叠面积图）
  - 积分消耗构成（环形图）
  - 主要成本动因排行（条形图）

##### 💰 消费分析

**积分总览（审计视角）**
- KPI卡片：总消耗积分、日均消耗、最高单日消耗、消费资源种类数
- 积分消耗与活跃关联图
- 按租户消耗排行

**Token消费分析**
- KPI卡片：Token总消耗积分、总消耗Token、输入/输出Token比例、平均CPM
- Token成本与用量关联趋势
- 输入/输出Token构成
- 模型消耗分析（支持按积分/用量切换）

**设备时长消费分析**
- KPI卡片：设备总消耗积分、云电脑/云手机使用时长、平均会话时长
- 设备时长消耗趋势（对比）
- 使用与抵扣趋势（对比）
- 分时段使用热力图
- 会话时长分布（对比）

##### 🔧 资产与使用分析

**整体使用总览**
- KPI卡片：总调用/运行次数、最活跃资产类型、总资产数、资产使用率
- 资产使用趋势（多类型对比）
- 资产创建 vs 使用分布

**资源类型切换器**
- 智能体：创建/调用趋势、Token消耗排行、明细列表
- 工作流：创建/运行趋势、积分消耗排行、明细列表
- 工具：调用频率、被引用来源分布、明细列表
- 工具库：工具数量排行、明细列表
- 知识库：容量趋势、明细列表
- 空间：资产分布、明细列表

##### 📝 消费明细

- **高级筛选器**: 资源类型（多选）、行为（搜索）、变动类型、时间范围
- **数据表格**: 日期、资源类型、行为、用量、消耗积分、计费租户、关联资产
- **列排序**: 支持多列排序
- **分页器**: 20/50/100/200条每页
- **导出CSV**: 一键导出当前筛选结果

##### 🏢 加入的租户

- **操作栏**: 添加到租户、批量移出租户、搜索
- **租户列表**: 可勾选、显示租户详细信息
- **移出租户**: 单个/批量移出（所有者不可移除，带提示）
- **添加租户弹窗**: 搜索+多选，已加入租户自动禁用

## 🛠 技术栈

### 前端框架
- **Vue 3**: 渐进式JavaScript框架
- **Element Plus 2.4.4**: 基于Vue 3的组件库

### 数据可视化
- **ECharts 5.4.3**: 强大的图表库
  - 活跃度日历热力图
  - 堆叠面积图
  - 水平条形图
  - 环形图/饼图
  - 组合图（柱状+折线）
  - 分时段热力图
  - 分组直方图

### 工具库
- **Day.js**: 轻量级日期处理库
- **自定义工具函数**: 日期格式化、数字格式化、数据聚合、导出CSV等

### 设计系统
- **CSS变量系统**: 统一的颜色、间距、阴影、圆角、过渡动画
- **响应式设计**: 支持1440px/1200px/768px断点
- **微交互动画**: Hover效果、Loading状态、过渡动画

## 🚀 快速开始

### 使用方法

1. **克隆或下载项目**
   ```bash
   cd 用户管理原型V3
   ```

2. **使用本地服务器运行**
   
   由于使用了ES6模块和跨域加载，建议使用本地服务器运行：

   **方法1: 使用Python**
   ```bash
   # Python 3
   python -m http.server 8080
   
   # Python 2
   python -m SimpleHTTPServer 8080
   ```

   **方法2: 使用Node.js (http-server)**
   ```bash
   npx http-server -p 8080
   ```

   **方法3: 使用VS Code Live Server**
   - 安装Live Server扩展
   - 右键点击index.html
   - 选择"Open with Live Server"

3. **访问系统**
   
   在浏览器中打开：http://localhost:8080/index.html

## 📊 Mock数据说明

系统内置完整的Mock数据生成器，自动生成真实感的测试数据：

- **60个用户**: 包含完整的个人信息、联系方式、状态
- **15个租户**: 包含不同版本（个人版/团队版/企业版）
- **各类资产**: 智能体、工作流、工具、工具库、知识库、空间
- **消费记录**: 最近6个月，每用户30-80条真实消费记录
- **聚合数据**: 按日/周/月聚合，支持图表展示

所有数据关系符合业务逻辑，时间序列真实，数值范围合理。

## 🌟 核心功能演示

### 用户列表页

1. 打开系统后默认显示用户列表
2. 顶部KPI卡片展示关键指标
3. 使用搜索栏筛选用户
4. 点击用户名称进入详情页

### 用户详情页

1. 从列表页点击用户名称进入
2. 顶部显示用户基础信息
3. 切换5个标签页查看不同维度的数据
4. 数据总览：查看活跃度和成本分析
5. 消费分析：深入了解积分、Token、设备使用情况

7. 消费明细：筛选并导出消费记录
8. 租户管理：添加/移出租户

### 数据可视化

- 所有图表支持悬浮查看详细数据
- 支持图例点击切换显示/隐藏系列
- 图表自动响应窗口大小调整
- 支持鼠标滚轮缩放（部分图表）

### 数据操作

- **复制功能**: 点击复制图标一键复制ID/手机/邮箱等
- **排序功能**: 点击表头排序
- **筛选功能**: 多维度筛选数据
- **导出功能**: CSV格式导出数据
- **分页功能**: 支持自定义每页显示数量
