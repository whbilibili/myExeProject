# 用户管理系统 - 高保真原型

## 📋 项目简介

这是一个基于 Vue 3 + Element Plus + ECharts 构建的企业级用户管理系统高保真原型。该原型完整实现了用户管理的核心功能，包含用户列表、用户详情（6个标签页）、完整的数据可视化和交互功能。

## ✨ 核心特性

### 技术栈
- **Vue 3** (Composition API) - 现代化响应式前端框架
- **Element Plus** - 企业级 UI 组件库
- **ECharts 5** - 专业级数据可视化库
- **Day.js** - 轻量级日期处理库
- **单 HTML 文件 SPA 架构** - 通过 CDN 引入，无需构建，开箱即用

- ✅ 列排序功能
- ✅ 用户操作（重置密码、禁用/启用、注销）
- ✅ 模拟数据：60+ 用户，完整真实感

## 🚀 快速开始

### 方法一：直接打开（推荐）

1. 直接用浏览器打开 `index.html` 文件
2. 无需任何安装或配置
3. 所有依赖通过 CDN 加载

### 方法二：本地服务器

如果遇到 CDN 加载问题，可以使用本地服务器：

```bash
# Python 3
python -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```

然后访问：http://localhost:8080

## 🎮 使用指南

### 用户列表页操作

1. **搜索用户**
   - 在搜索框输入关键词（支持姓名、ID、手机、邮箱、MIS）
   - 选择状态筛选（正常/已禁用）
   - 选择租户筛选（支持多选）
   - 选择日期范围

2. **查看用户详情**
   - 点击用户名称进入详情页

3. **用户操作**
   - 重置密码：点击"重置密码"按钮
   - 禁用/启用：点击对应按钮并确认
   - 注销：点击"注销"按钮，需要输入"确认注销"

### 用户详情页操作

1. **全局筛选**
   - 选择时间范围（支持快捷选项）
   - 选择租户（筛选特定租户的数据）

2. **数据总览**
   - 查看 KPI 卡片
   - 切换图表聚合粒度（按日/按周/按月）
   - 查看各类资源的消耗分析

3. **消费明细**
   - 使用高级筛选器
   - 导出数据为 CSV

4. **资产明细**
   - 切换不同资产类型标签
   - 刷新数据

5. **租户管理**
   - 添加用户到新租户
   - 移出用户（单个或批量）

## 🔧 自定义和扩展

### 修改样式

编辑 `assets/css/custom.css` 文件，修改 CSS 变量：

```css
:root {
  --primary-color: #409EFF;  /* 主色 */
  --success-color: #67C23A;  /* 成功色 */
  /* ... 更多变量 */
}
```

### 添加新数据

编辑 `assets/js/mock-data.js` 文件，修改数据生成逻辑：

```javascript
// 修改用户数量
const users = generateUsers(100, tenants);  // 生成 100 个用户

// 修改消费记录数量
const recordCount = utils.randomInt(50, 100);  // 每用户 50-100 条
```

### 添加新图表

1. 在 `assets/js/charts.js` 添加图表配置
2. 在对应组件中调用配置
3. 使用 ECharts 实例渲染
