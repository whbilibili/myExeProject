# Web Note Helper 功能规划

本文档概述了 Web Note Helper 浏览器插件的未来发展计划和功能路线图。

## 当前版本 (v1.0)

当前版本实现了以下核心功能：

- 在任何网页上添加文本笔记
- 将笔记与当前网页URL关联
- 查看当前网站的笔记和所有笔记
- 删除不需要的笔记
- 本地存储所有笔记数据

## 近期计划 (v1.x)

以下功能计划在未来的小版本更新中实现：

- [ ] 笔记编辑功能
