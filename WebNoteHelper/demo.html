如何使用 Web Note Helper

  1. 点击浏览器工具栏中的 Web Note Helper 图标
  2. 在弹出的窗口中，你会看到当前网站的URL
  3. 在文本框中输入你的笔记内容
  4. 点击"保存笔记"按钮
  5. 你的笔记将会保存并显示在"当前网站笔记"和"所有笔记"部分

为什么使用 Web Note Helper

Web Note Helper 可以帮助你：

  • 在浏览网页时快速记录重要信息
  • 将笔记与特定网站关联，方便日后查找
  • 整理和管理你在不同网站上的笔记
  • 提高你的网页浏览和研究效率

隐私保护

Web Note Helper 尊重你的隐私：

  • 所有笔记数据仅保存在你的浏览器本地存储中
  • 不会将任何数据发送到外部服务器
  • 只请求必要的权限来实现其功能
